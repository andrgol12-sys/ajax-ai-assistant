# 🚚 Ajax AI Assistant

AI-помощник для транспортной компании на базе Telegram и GigaChat API.

Проект создан в рамках изучения Cursor AI и подхода вайб-кодинга. Бот консультирует по вопросам логистики, грузоперевозок, рефрижераторного транспорта, ставок на перевозку и маршрутов.

---

## ✨ Возможности

- 🤖 Ответы через GigaChat API

- 🚚 Специализация на логистике и грузоперевозках

- 🧠 Память последних сообщений пользователя

- 🔄 Команда `/reset` для очистки истории диалога

- 🔐 Хранение секретных данных через `.env`

- 📝 Подробное логирование работы

- 💬 Поддержка контекста диалога

---

## 📌 Пример использования

Пользователь:

> Какая средняя ставка на перевозку рефрижераторной фурой 20 тонн по маршруту Москва — Санкт-Петербург?

Бот:

> Средняя ставка составляет ориентировочно 70–90 тыс. рублей в зависимости от груза, температурного режима, сезона и рыночной конъюнктуры.

Пользователь:

> А по маршруту Санкт-Петербург — Москва?

Бот понимает контекст предыдущего сообщения и продолжает диалог без необходимости повторного ввода параметров.

## ⚙️ Установка

### 1. Клонирование репозитория

```bash

git clone [https://github.com/YOUR_USERNAME/ajax-ai-assistant.git](https://github.com/YOUR_USERNAME/ajax-ai-assistant.git)

cd ajax-ai-assistant

```

### 2. Создание виртуального окружения

```bash

python -m venv .venv

```

### 3. Активация окружения

Windows PowerShell:

```powershell

.venv\Scripts\Activate.ps1

```

### 4. Установка зависимостей

```bash

pip install -r requirements.txt

```

---

## 🔑 Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```env

TELEGRAM_BOT_TOKEN=your_telegram_bot_token

GIGACHAT_AUTHORIZATION_KEY=your_gigachat_authorization_key

GIGACHAT_SCOPE=GIGACHAT_API_PERS

GIGACHAT_MODEL=GigaChat

GIGACHAT_VERIFY_SSL=false

```

## 🧠 Контекст диалога

Бот хранит историю последних сообщений каждого пользователя и использует её при формировании новых запросов к GigaChat.

Это позволяет:

- поддерживать контекст беседы;

- понимать уточняющие вопросы;

- использовать информацию из предыдущих сообщений;

- вести более естественный диалог.

Для очистки памяти используется команда:

```text

/reset

```
